# Buyer Support Boundaries

Whole Home Harvest is sold as a digital website asset, not as an ongoing managed service.

## Included with sale

- Source code
- Documentation
- Deployment guide
- Transfer checklist
- Monetization guide
- Buyer Control Center
- QA checklist

## Not included unless separately agreed

- Ongoing hosting management
- Custom coding after sale
- Buyer domain purchase
- Buyer email platform setup
- Buyer AI API billing
- Buyer affiliate approvals
- Buyer ad network approvals
- Buyer analytics account setup
- Content writing after sale
- Legal, tax, financial, agricultural, food-safety, or medical advice

## Recommended support window

A reasonable handoff support window is 3 to 7 days for clarification on files and deployment instructions only.

## Recommended wording for Flippa

Buyer receives the complete source code and documentation package. The platform is designed for easy transfer and buyer customization. Buyer is responsible for connecting their own third-party accounts, domain, analytics, email provider, ad network, affiliate accounts, and AI API keys.
