# Buyer Control Center Guide

The Buyer Control Center is available at:

`/admin-config`

It lets the buyer organize the most important setup details after purchase.

## What it controls

- Business name
- Tagline
- Domain
- Contact email
- Brand colors
- AI provider notes
- Email/newsletter notes
- AdSense publisher ID
- Analytics ID
- Gumroad product URL
- Affiliate tag
- Social links
- Lead magnet title

## Important note

This version saves configuration in the browser using local storage. It is a safe buyer handoff tool and does not store secrets on a server.

For a production admin system, the buyer can connect a backend such as Supabase, Firebase, or their own CMS.

## Why this helps the buyer

Instead of searching through the codebase after purchase, the buyer has one clear place to understand what needs to be replaced, connected, or customized.
