# Seller Handoff Guide — Whole Home Harvest

This file is designed to protect the seller and make the buyer transfer simple.

## What the buyer receives

- Full React + Vite source code
- GitHub repository access or ZIP export
- Cloudflare deployment instructions
- Buyer Quick Start Guide
- Monetization Guide
- QA Checklist
- Transfer Checklist
- Roadmap
- Buyer Control Center at `/admin-config`
- Flippa listing copy
- Social media posts
- SEO keyword list
- Email welcome sequence

## What the seller should transfer

1. GitHub repository ownership or collaborator access.
2. The final source-code ZIP.
3. The live Cloudflare Pages URL or instructions to redeploy in the buyer account.
4. Any included documentation files.
5. Any optional screenshots, logos, or listing assets that were part of the sale.

## What the buyer must connect themselves

The buyer is responsible for connecting and paying for their own:

- Custom domain
- Email marketing provider
- AI API key/provider
- Gumroad or payment account
- Affiliate accounts
- AdSense/display ad account
- Analytics account
- Social media accounts
- Any paid third-party services

## Recommended seller boundary

After delivery, point the buyer to the documentation folder and the Buyer Control Center. Do not offer open-ended custom development unless it is separately paid.

Suggested sentence:

> The asset is delivered with source code, documentation, deployment instructions, and setup guides. Buyer is responsible for connecting their own accounts and customizing the platform after transfer.
