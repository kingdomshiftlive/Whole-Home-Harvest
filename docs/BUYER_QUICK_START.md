# Buyer Quick Start — Whole Home Harvest

## 1. Run the site locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## 2. Deploy to Cloudflare Pages

Use:

```text
Build command: npm run build
Output directory: dist
```

If the project is inside a subfolder, set the root directory to that subfolder.

## 3. Update owner details

Replace the default email address with your business email in:

- `src/data/siteData.js`
- `src/pages/Contact.jsx`
- `src/pages/Legal.jsx`

## 4. Connect monetization

- Add affiliate links in `src/data/siteData.js`.
- Add AdSense details after approval.
- Connect your email provider.
- Add analytics.

## 5. Review legal pages

The included legal pages are starter templates. Review them for your business, location, and monetization setup.
