# Seller Final Checklist Before Publishing on Flippa

## Must be complete

- [ ] Live Cloudflare site opens
- [ ] GitHub repository is clean
- [ ] `npm run build` passes
- [ ] Release ZIP is created
- [ ] Screenshots are captured
- [ ] Flippa listing copy is ready
- [ ] Buyer transfer terms are clear
- [ ] No personal API keys are included
- [ ] No personal analytics accounts are included
- [ ] No personal affiliate IDs are promised as included
- [ ] Buyer understands they connect their own accounts

## Links to include in private buyer handoff

- GitHub repository
- Cloudflare Pages project or deployment instructions
- Release ZIP
- Documentation folder
- Buyer Control Center URL

## Seller reminder

Do not promise income, traffic, ad approvals, affiliate approvals, or custom support unless separately agreed in writing.
