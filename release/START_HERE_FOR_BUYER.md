## Recommended first steps

1. Install dependencies with `npm install`.
2. Run locally with `npm run dev`.
3. Build with `npm run build`.
4. Deploy to Cloudflare Pages or your preferred static host.
5. Visit `/admin-config` to record your business, brand, AI, analytics, email, ads, affiliate, and digital product settings.
6. Replace placeholder accounts with your own third-party accounts.

## Important

Do not place private API keys directly into public frontend code. Use a secure backend or serverless function for production AI integrations.
